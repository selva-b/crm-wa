# Wazelo CRM — Google Flow Labs Video Prompts

**How to use:**
1. Open [Google Flow Labs](https://labs.google/flow)
2. Paste each CLIP prompt into the prompt box → Click Generate
3. After generating a clip, click the **[+] Extend** button to chain the next clip
4. Max per generation: **8 seconds** — use Extend to build longer videos
