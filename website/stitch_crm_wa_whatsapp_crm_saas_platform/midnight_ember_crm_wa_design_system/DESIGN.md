# Midnight Ember Design System — CRM-WA

## Color Palette
- **Primary (Orange Amber):** #d97706
- **Primary Container:** #f59e0b
- **Surface (Dark BG):** #0f1117
- **Surface Container:** #1a1d2e
- **Surface Elevated:** #2d3044
- **On-Surface (Text):** #e8eaed
- **On-Surface-Variant (Secondary Text):** #9ca3b4
- **Error:** #f87171
- **Success:** #34d399
- **Warning:** #fbbf24
- **Info:** #60a5fa
- **Border:** #2d3044
- **Sidebar BG:** #0d0f1a
- **Header BG:** #0f1117

## Typography
- Font: Inter (all weights)
- Page titles: 20px / 600
- Section headers: 16px / 600
- Body: 14px / 400
- Labels/captions: 12px / 400

## Layout Pattern
- Fixed sidebar (240px) + fixed header (64px) + scrollable main
- Intercom/Slack-inspired dark minimal aesthetic
- Sidebar: dark (#0d0f1a), icons + labels, amber active state
- Cards: #1a1d2e bg, 8px radius, 1px #2d3044 border
- Buttons: amber primary, surface-elevated secondary
- Status badges: pill shape, colored by status
- Tables: dark rows, subtle hover
- Inputs: dark bg, amber focus ring
