Configuring Webhook Endpoints

To receive real-time updates for WhatsApp messages, you must configure a Webhook Endpoint. This allows our servers to push data to your application as soon as an event occurs.

Prerequisites

  • A publicly accessible HTTPS URL.
  • Verification Token (defined in your CRM settings).
  • The ability to handle a GET request for validation.
POST /webhooks/callback
Content-Type: application/json

{
  "object": "whatsapp_business_account",
  "entry": [...]
}

Once you've set up your listener, go to Settings > API > Webhooks to enter your endpoint URL and save changes.
